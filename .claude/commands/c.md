Commit and push all changes:

1. Run `git status` and `git diff` to see changes
2. Stage all changes with `git add -A`
3. Create a concise commit message summarizing the changes
4. Commit with co-author: `Co-Authored-By: Claude <noreply@anthropic.com>`
5. Push to remote

If no changes exist, say so. Warn about any sensitive files before committing.
